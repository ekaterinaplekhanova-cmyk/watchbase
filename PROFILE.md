# Контент-маркетолог

Привет! Я создаю и развиваю контент для премиального e-commerce.

## ✨ Чем занимаюсь

- Создаю товарные карточки, SEO-описания и посты для социальных сетей.
- Автоматизирую рутинные задачи контент-менеджмента.
- Контролирую актуальность ассортимента и качество публикуемых материалов.
- Работаю с аналитикой: отслеживаю трафик, конверсии и поведение пользователей.

## 🛠 Ключевые навыки

- Генерация продуктового контента на основе данных из открытых источников.
- Проверка технических характеристик товаров премиального сегмента.
- Работа с LLM: промпт-инжиниринг, пост-редактирование, контроль фактов.
- Публикация и ведение контента в социальных сетях: ВКонтакте, Дзен.
- SEO-оптимизация статей и товарных страниц.
- Аналитика сайтов через Ройстат и Яндекс.Метрику.

## 💻 Стек

- **Языки:** Python, JavaScript
- **Интерфейсы:** Streamlit
- **LLM / ML:** Ollama, `qwen2.5:14b`, промпт-инжиниринг
- **Парсинг и автоматизация:** requests, BeautifulSoup, Playwright
- **CMS:** 1С-Битрикс, Tilda
- **Аналитика:** Яндекс.Метрика, Ройстат

## 🎯 В фокусе

Точность, скорость и качество контента для премиальных брендов. Люблю порядок в данных и красивый результат на выходе.
